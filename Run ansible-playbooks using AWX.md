# Run ansible playbooks using AWX 
* In this repo will see how to run ansible playbooks on remote machine using AWX Web UI.

# Prerequisites
* A running AWX Web UI machine.
* A remote AWS amazon ec2 instance.
* A Key pair (pem format)
